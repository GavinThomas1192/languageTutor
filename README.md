* To get up and running...

  * Git clone...
  * `yarn` || `npm install`
  * `cd languageTutor`
  * `yarn start` || `npm run start`

- To run tests...

  * `yarn test` || `npm run test`

* To use an .env file...
  * At the root of your directory...
    * `touch .env`
    * Inside .env
      * REACT_APP_SOME_RANDOM_KEY='someRandomKey'
        \*\* Note: you must use REACT_APP
